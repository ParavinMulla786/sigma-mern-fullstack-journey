📘 JavaScript Notes

What is JavaScript?

JavaScript is a lightweight, object-oriented scripting programming language used to make web pages interactive.

It is one of the three core technologies of web development: HTML, CSS, and JavaScript.

Scripting Language

A scripting language is interpreted at runtime and does not require compilation.

Examples: JavaScript, Python, PHP

Uses of JavaScript

  • Form Validation
  • Animations
  • Dynamic Content
  • Web Applications
  • Games

History

JavaScript was created by Brendan Eich in 1995 at Netscape. Originally called LiveScript.

Ways to Add JavaScript

Inline:

<button onclick="alert('Hello')">Click</button>

Internal:

<script> alert("Hello"); </script>

External:

<script src="script.js"></script>
