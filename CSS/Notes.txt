CSS – BOX MODEL

The CSS Box Model defines how elements are displayed on a webpage and how their width and height are calculated.

Every HTML element is treated as a rectangular box.

The box model has four main components:

Content
The content is the actual data inside the element.
It can be text, image, video, etc.
The content area is controlled using width and height properties.

Example:
div {
width: 200px;
height: 100px;
}

Padding
Padding is the space between the content and the border.
It creates space inside the element.
Padding is transparent.

Example:
div {
padding: 20px;
}

You can also set padding individually:
padding-top
padding-right
padding-bottom
padding-left

Border
The border surrounds the content and padding.
It gives a visible outline to the element.

Example:
div {
border: 2px solid black;
}

Margin
Margin is the space outside the border.
It creates space between elements.

Example:
div {
margin: 20px;
}

CALCULATING TOTAL WIDTH AND HEIGHT

Total Width =
Width

Left Padding + Right Padding

Left Border + Right Border

Left Margin + Right Margin

Total Height =
Height

Top Padding + Bottom Padding

Top Border + Bottom Border

Top Margin + Bottom Margin

By default:
box-sizing: content-box;

If we use:
box-sizing: border-box;

Then padding and border are included inside the width and height.

CSS DISPLAY PROPERTY

The display property specifies how an element is shown on a webpage.

Every HTML element has a default display value.
Most common default values are block and inline.

INLINE ELEMENTS

An inline element:

Does not start on a new line

Takes only required width

Appears in the same line

Examples:
span
a
img

Example:

<p>This is <span>inline</span> text.</p>

BLOCK ELEMENTS

A block element:

Starts on a new line

Takes full width available

Examples:
div
p
h1 to h6

INLINE-BLOCK

display: inline-block;

Stays in the same line

Allows width and height like block elements
